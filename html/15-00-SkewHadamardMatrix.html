skew Hadamard matrix


A Hadamard matrix H is skew Hadamard if H+HT=2⁢I.

A collection of skew Hadamard matrices, including at least one example of every order n≤100 and also including every equivalence class of order ≤28, is available http://www.rangevoting.org/SkewHad.htmlat this web page. It has been conjectured that one exists for every positive order divisible by 4.

Reid and Brown in 1972 showed that there exists a “doubly regular tournament of order n” if and only if there exists a skew Hadamard matrix of order n+1.
