there exist additive functions which are not linear


Example 1.

There exists a function f:ℝ→ℝ which is additive but not linear.

Proof.

Let V be the infinite dimensional vector space ℝ over the field ℚ. Since 1 and 2 are two independent vectors in V, we can extend the set {1,2} to a basis E of V (notice that here the axiom of choice is used).

Now we consider a linear function f:V→ℝ such that f⁢(1)=1 while f⁢(e)=0 for all e∈E∖{1}. This function is ℚ-linear (i.e. it is additive on ℝ) but it is not ℝ-linear because f⁢(2)=0≠2⁢f⁢(1). ∎
