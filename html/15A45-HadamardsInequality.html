Hadamard’s inequality


Let A=(ai⁢j) with 1≤i,j≤n∈ℕ be a square matrix with complex coefficients. Then the following inequality holds:

|det⁡(A)|≤∏i=1n(∑j=1n|ai⁢j|2)12.

Moreover, if A is Hermitian and positive semidefinite, the following inequality holds:

det⁡(A)≤∏i=1nai⁢i,

with equality if and only if A is a diagonal matrix.
