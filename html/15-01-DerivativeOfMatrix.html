derivative of matrix


Suppose I is an open set of ℝ, and for each t∈I, A⁢(t) is an n×m matrix. If each element in A⁢(t) is a differentiable function of t, we say that A is a differentiable, and define the derivative of A componentwise. This derivative we shall write as dd⁢t⁢A or d⁢Ad⁢t.

Properties

In the below we assume that all matrices are dependent on a parameter t and the matrices are differentiable with respect to t.

  1. 1.

    For any n×m matrix A,

    (d⁢Ad⁢t)T = dd⁢t⁢(AT),

    where T is the matrix transpose.

  2. 2.

    If A⁢(t),B⁢(t) are matrices such that A⁢B is defined, then

    dd⁢t⁢(A⁢B)=d⁢Ad⁢t⁢B+A⁢d⁢Bd⁢t.
  3. 3.

    When A⁢(t) is invertible,

    dd⁢t⁢(A-1)=-A-1⁢d⁢Ad⁢t⁢A-1.
  4. 4.

    For a square matrix A⁢(t),

    tr⁡(d⁢Ad⁢t) = dd⁢t⁢tr⁡(A),

    where tr is the matrix trace.

  5. 5.

    If A⁢(t),B⁢(t) are n×m matrices and A∘B is the Hadamard product of A and B, then

    dd⁢t⁢(A∘B)=d⁢Ad⁢t∘B+A∘d⁢Bd⁢t.
