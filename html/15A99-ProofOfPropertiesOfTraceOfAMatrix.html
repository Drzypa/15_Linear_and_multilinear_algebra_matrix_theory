proof of properties of trace of a matrix


Proof of Properties:

  1. 1.

    Let us check linearity. For sums we have

    trace⁡(A+B) = ∑i=1n(ai,i+bi,i)   (property of matrix addition)
    = ∑i=1nai,i+∑i=1nbi,i⁢(property of sums)
    = trace⁡(A)+trace⁡(B).

    Similarly,

    trace⁡(c⁢A) = ∑i=1nc⋅ai,i⁢(property of matrix scalar multiplication)
    = c⋅∑i=1nai,i⁢(property of sums)
    = c⋅trace⁡(A).
  2. 2.

    The second property follows since the transpose does not alter the entries on the main diagonal.

  3. 3.

    The proof of the third property follows by exchanging the summation order. Suppose A is a n×m matrix and B is a m×n matrix. Then

    trace⁡A⁢B = ∑i=1n∑j=1mAi,j⁢Bj,i
    = ∑j=1m∑i=1nBj,i⁢Ai,j⁢(changing summation order)
    = trace⁡B⁢A.
  4. 4.

    The last property is a consequence of Property 3 and the fact that matrix multiplication is associative;

    trace⁡(B-1⁢A⁢B) = trace⁡((B-1⁢A)⁢B)
    = trace⁡(B⁢(B-1⁢A))
    = trace⁡((B⁢B-1)⁢A)
    = trace⁡(A).
