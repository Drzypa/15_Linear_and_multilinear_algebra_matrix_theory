Farkas lemma, proof of


We begin by showing that at least one of the systems has a solution.

Suppose that system 2 has no solution. Let S be the cone in ℝn generated by nonnegative linear combinations of the rows a1,…,am of A. The set S is closed and convex. Since system 2 is unsolvable, the vector c is not in S; therefore, there exist a scalar α and n-column vector x such that the hyperplane xT⁢v=α separates c from S in ℝn. This hyperplane can be selected so that for any point v∈S,

xT⁢cT>α>xT⁢vT.

Since 0∈S, this implies that α>0. Hence for any w≥0,

α>xT⁢(w⁢A)T=w⁢A⁢x=∑i=0mwi⁢(A⁢x)i.

Each (A⁢x)i is nonpositive. Otherwise, by selecting w with wi sufficiently large and all other wj=0, we would get a contradiction. We have now shown that x satisfies A⁢x≤0 and c⁢x=(c⁢x)T=xT⁢cT>α>0, which means that x is a solution of system 1. Thus at least one of the systems is solvable.

We claim that systems 1 and 2 are not simultaneously solvable. Suppose that x is a solution of system 1 and w is a solution of system 2. Then for each i, the inequality wi⁢(A⁢x)i≤0 holds, and so w⁢(A⁢x)≤0. However,

(w⁢A)⁢x=c⁢x>0,

a contradiction. This completes the proof.
