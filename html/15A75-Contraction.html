contraction


Definition Let ω be a smooth k-form on a smooth manifold M, and let ξ be a smooth vector field on M. The contraction of ω with ξ is the smooth (k-1)-form that maps x∈M to ωx⁢(ξx,⋅). In other words, ω is point-wise evaluated with ξ in the first slot. We shall denote this (k-1)-form by ιξ⁢ω. If ω is a 0-form, we set ιξ⁢ω=0 for all ξ.

Properties Let ω and ξ be as above. Then the following properties hold:

  1. 1.

    For any real number k

    ιk⁢ξ⁢ω=k⁢ιξ⁢ω.
  2. 2.

    For vector fields ξ and η

    ιξ+η⁢ω = ιξ⁢ω+ιη⁢ω,
    ιξ⁢ιη⁢ω = -ιη⁢ιξ⁢ω,
    ιξ⁢ιξ⁢ω = 0.
  3. 3.

    Contraction is an anti-derivation [1]. If ω1 is a p-form, and ω2 is a q-form, then

    ιξ⁢(ω1∧ω2)=(ιξ⁢ω1)∧ω2+(-1)p⁢ω1∧(ιξ⁢ω2).
