Hamel function


A function h:ℝn→ℝ is said to be a Hamel function if h, considered as a subset {(x,h(x)}⊂ℝn+1, is a Hamel basis for ℝn+1 over ℚ. We denote the set of n-dimensional Hamel function by H⁢F⁢(ℝn).
