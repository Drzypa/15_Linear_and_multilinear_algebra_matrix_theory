Frobenius theorem on linear determinant preservers


Let 𝔽 be an arbitrary field. Consider ℳn⁢(𝔽), the vector space of all n×n matrices over 𝔽. Let 𝒢⁢ℒn⁢(𝔽) be the set of all nonsingular matrices P∈ℳn⁢(𝔽).

Definition 1.

A linear endomorphism φ:Mn⁢(F)⟶Mn⁢(F) is said to be in standard form, if either ∃P,Q∈G⁢Ln⁢(F)⁢∀A∈Mn⁢(F):φ⁢(A)=P⁢A⁢Q or ∃P,Q∈G⁢Ln⁢(F)⁢∀A∈Mn⁢(F):φ⁢(A)=P⁢A⊤⁢Q.

The classical on linear preservers of the determinant function [GF] reads as follows.

Theorem 2.

If φ:Mn⁢(C)⟶Mn⁢(C) is a linear automorphism such that det⁡(φ⁢(A))=det⁡(A) for all A∈Mn⁢(C), then φ is in standard form with
det⁡(P⁢Q)=1.

It is well known that the can be strengthened.

Theorem 3.

Let F be an arbitrary field and let φ:Mn⁢(F)⟶Mn⁢(F) be a linear endomorphism. Then the following conditions are equivalent:
(i) det⁡(φ⁢(A))=det⁡(A) for all A∈Mn⁢(F), (ii) φ is in standard form with det⁡(P⁢Q)=1.

The above strengthened version of the can be derived from the Dieudonné theorem on linear preservers of the singular matrices.
