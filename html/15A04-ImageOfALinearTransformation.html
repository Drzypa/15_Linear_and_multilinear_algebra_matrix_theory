image of a linear transformation


Definition Let T:V→W be a linear transformation. Then the image of T is the set

Im⁡(T)={w∈W∣w=T⁢(v)⁢for some⁢v∈V}=T⁢(V).

0.0.1 Properties

  1. 1.

    The dimension of Im⁡(T) is called the rank of T;

  2. 2.

    T is a surjection, if and only if Im⁡(T)=W;

  3. 3.

    Im⁡(T) is a vector subspace of W;

  4. 4.

    If L:W→U is a linear transformation, then Im⁡(L⁢T)=L⁢(Im⁡(T));
